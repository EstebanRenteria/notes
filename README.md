# Gestor de Notas Elegante

Aplicación completa para gestión de notas con autenticación JWT 

## Características

- ✅ Autenticación JWT
- ✅ CRUD completo de notas
- 🖼️ Vista previa de adjuntos
- 🔍 Filtros avanzados
- 📱 Diseño responsivo

## Instalación

1. Clonar repositorio
    git clone
2. Instalar dependencias:
   npm install
3. Correr Proyecto:
   npm run dev
